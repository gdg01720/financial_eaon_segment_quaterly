# 日本語フォント設置フォルダ

このフォルダに日本語フォントファイルを配置してください。

## 推奨フォント

**IPAexゴシック (ipaexg.ttf)**

## ダウンロード方法

1. [IPAフォント公式サイト](https://moji.or.jp/ipafont/) にアクセス
2. 「IPAexフォント」をダウンロード
3. 解凍して `ipaexg.ttf` をこのフォルダにコピー

## フォルダ構成（配置後）

```
fonts/
├── README.md      # このファイル
└── ipaexg.ttf     # 日本語フォント
```

## 注意事項

- フォントファイルはGitリポジトリに含めてください
- Streamlit Cloudにデプロイする際もフォントファイルが必要です
- IPAフォントはIPAフォントライセンスv1.0に基づき無償で利用できます

## 代替フォント

IPAexゴシック以外にも、以下のフォントが使用可能です：

- IPAexMincho (ipaexm.ttf) - 明朝体
- Noto Sans CJK JP
- 源ノ角ゴシック

ただし、コード内のフォントファイル名を変更する必要があります。
